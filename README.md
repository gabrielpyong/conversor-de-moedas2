# Conversor de Moedas

Um conversor de moedas simples que converte entre Euro, Real, Yuan, Pesos Argentinos, Libras, Dólar e Bitcoin. Usa a ExchangeRate-API para moedas fiduciárias e a CoinGecko API para Bitcoin.
